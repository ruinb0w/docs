Popup 是一个简单的底部弹窗组件

用例 ​

vue
<script setup>
import Popup from "../../components/Popup/Popup.vue";

const showPopup = ref(false);
</script>

<template>
  <button @click="showPopup = true">显示</button>
  <Popup v-model="showPopup">弹窗内容</Popup>
</template>
