# 项目结构

```
src
|-app.controller.spec.ts
|-app.controller.ts
|-app.module.ts
|-app.service.ts
|-main.ts
```

| 文件                   | 描述             |
| ---------------------- | ---------------- |
| app.controller.ts      | 控制器           |
| app.controller.spec.ts | 控制器的单元测试 |
| app.module.ts          | 模块             |
| app.service.ts         | 服务             |
| main.ts                | 入口文件         |

## main.ts

main.ts 是项目的入口文件
